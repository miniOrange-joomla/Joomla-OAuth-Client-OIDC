# Login with Keycloak OAuth Single Sign-On (SSO) for Joomla

Simplify authentication on your Joomla site with the **Login with Keycloak OAuth Single Sign-On (SSO)** plugin. Enable seamless login using **Keycloak credentials**, making Joomla a trusted **OAuth 2.0 client** for improved user convenience and security.

## Key Highlights:
- **Effortless SSO**:  
  Smooth integration between Joomla and Keycloak, eliminating the need to remember multiple usernames and passwords.
- **Flexible Configuration**:  
  Fully customizable authentication flows to ensure compatibility with a wide range of Keycloak servers.

## Grant Types Supported:
- Authorization Code
- Implicit Grant
- Resource Owner Password Credentials
- Refresh Token
- PKCE (Proof Key for Code Exchange)

---

## Versions & Features:

### **Free Version**
- Login with Keycloak server credentials
- Auto-fill Keycloak server configuration
- Basic attribute mapping (Username, Email)
- Limited user authentication

### **Standard Version**
- Unlimited authentication and user registration
- Login widget customization
- Authorization Code Grant support
- Custom redirect URLs

### **Premium Version**
- Advanced attribute and group mapping
- Force authentication for enhanced security
- OpenID Connect (OIDC) support
- Domain-specific registration

### **Enterprise Version**
- Additional endpoints to retrieve user groups from Keycloak
- Advanced grant settings
- Detailed login reports and analytics

---

## Add-Ons:
- **Keycloak User Sync**: Synchronize and provision users between Joomla and Keycloak.
- **SCIM User Provisioning**: Sync user creation, updates, and deletion from IDP to Joomla.
- **Role-Based Redirection**: Redirect users based on received roles or groups.
- **SSO Session Management**: Manage session durations for enhanced security.
- **Two-Factor Authentication**, **Login Audit**, and more.

---

## Quick Installation:
1. Go to **Extensions → Manage → Install**
2. Search for **"miniOrange OAuth Client Keycloak"** and install the plugin.
3. Configure under **Component → miniOrange OAuth Client → Configure OAuth**
